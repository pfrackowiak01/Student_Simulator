# Student Simulator
Mobile game project for my studies in Android Studio (Java).

---

It's a clicker with extra features:
- ListView
- ArrayAdapter
- ActivityResultLauncher
- BackgroundService/ForegroundService
- SQLite
- GPS
- Fragments
- Countdown timer
- Implicit intent (taking a photo)
